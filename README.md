## бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ–бһ‘бһјбһ‘бҹ…бһ“бҹғбһӮбһҳбҹ’бһҡбҹ„бһ„ (Project Overview)
бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ·бһҹбҹ’бһҹ (SMS) бһҠбҹҸбһ‘бһјбһӣбҹҶбһ‘бһјбһӣбһ¶бһҷбһҳбһҪбһҷ бһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбһҖбһҮбһ¶бһҳбһҪбһҷбһ“бһјбһңбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢбһҳбһ»бһҒ (Face Recognition) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһңбһҸбҹ’бһҠбһҳбһ¶бһ“бҹ” бһңбһ¶бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбһ·бһҹбҹ’бһҹ, бһҠбҹҒбһ”бҹүбһ¶бһҸбһәбһҳбҹүбһ„бҹӢ, бһҮбҹҶбһ“бһ¶бһү, бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбҹҖбһ“, бһҶбҹ’бһ“бһ¶бҹҶбһҹбһ·бһҖбҹ’бһҹбһ¶, бһҳбһ»бһҒбһңбһ·бһҮбҹ’бһҮбһ¶, бһҖбһ¶бһӣбһңбһ·бһ—бһ¶бһӮ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһңбһҸбҹ’бһҸбһҳбһ¶бһ“бҹ” бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ GUI бһ‘бҹҶбһ“бһҫбһ”бһҡбһ”бһҹбҹӢ **Tkinter**, бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ **MySQL**, бһҖбһ¶бһҡбһ“бһ¶бҹҶбһ…бһјбһӣ/бһ“бһ¶бҹҶбһ…бҹҒбһүбһҜбһҖбһҹбһ¶бһҡ **Excel**, бһ“бһ·бһ„бһҖбһ¶бһҡбһҗбһҸбһҡбһјбһ”бһ—бһ¶бһ–бһҸбһ¶бһҳбһҡбһҷбҹҲ Webcamбҹ”

## бһӣбһҖбҹ’бһҒбһҺбҹҲбһ–бһ·бһҹбҹҒбһҹбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ— (Main Features)
- бһ”бһ“бҹ’бһҗбҹӮбһҳ, бһ’бҹ’бһңбһҫбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ—бһ¶бһ–, бһӣбһ»бһ”, бһ“бһ·бһ„бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһ·бһҹбҹ’бһҹ
- бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҠбҹҒбһ”бҹүбһ¶бһҸбһәбһҳбҹүбһ„бҹӢ, бһҮбҹҶбһ“бһ¶бһү, бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбҹҖбһ“, бһҶбҹ’бһ“бһ¶бҹҶбһҹбһ·бһҖбҹ’бһҹбһ¶ бһ“бһ·бһ„бһҳбһ»бһҒбһңбһ·бһҮбҹ’бһҮбһ¶
- бһҖбҹҶбһҺбһҸбҹӢбһҖбһ¶бһӣбһңбһ·бһ—бһ¶бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶
- бһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһңбһҸбҹ’бһҸбһҳбһ¶бһ“ (бһҠбҹ„бһҷбһҠбҹғ бһ“бһ·бһ„бһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһҹбҹ’бһҖбҹҒбһ“бһҳбһ»бһҒ)
- бһ“бһ¶бҹҶбһ…бһјбһӣ/бһ“бһ¶бҹҶбһ…бҹҒбһүбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҜбһҖбһҹбһ¶бһҡ Excel
- бһҳбһҫбһӣ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһңбһҸбҹ’бһҸбһҳбһ¶бһ“

## бһҡбһ”бҹҖбһ”бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ (How to Use)

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ“бҹғбһҜбһҖбһҹбһ¶бһҡбһҖбһјбһҠбһ“бһёбһҳбһҪбһҷбҹ— (Purpose of Each Code File)

* **`main_app.py`**: бһҮбһ¶бһ…бҹҶбһҺбһ»бһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё (Main entry point)бҹ” бһңбһ¶бһ”бһ„бҹ’бһҖбһҫбһҸ GUI, бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҳбҹүбһјбһҢбһ»бһӣбһ“бһ¶бһ“бһ¶ бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ”
* **`ui_components.py`**: бһҡбһ¶бһӣбҹӢбһҹбһҳбһ¶бһҹбһ’бһ¶бһҸбһ» Tkinter UI, бһ”бҹ’бһӣбһ„бҹӢ (Layouts), бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһ”бҹҗбһ‘бҹ’бһҳ (Styles)бҹ”
* **`student_ops.py`**: бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ CRUD бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ·бһҹбҹ’бһҹ, бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ‘бһҳбҹ’бһҡбһ„бҹӢ (Form), бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ Treeviewбҹ”
* **`db_utils.py`**: бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ Database, бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸ Schema, бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ· Query, бһ“бһ·бһ„бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бһҺбҹ’бһҸбҹ„бҹҮбһўбһ¶бһҹбһ“бҹ’бһ“ (Caching)бҹ”
* **`excel_utils.py`**: бһҖбһ¶бһҡбһ“бһ¶бҹҶбһ…бһјбһӣ/бһ“бһ¶бҹҶбһ…бҹҒбһү бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһ·бһҹбҹ’бһҹ бһ“бһ·бһ„бһңбһҸбҹ’бһҸбһҳбһ¶бһ“ бһ–бһё/бһ‘бҹ… бһҜбһҖбһҹбһ¶бһҡ Excelбҹ”
* **`camera_utils.py`**: бһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ Webcam, бһҖбһ¶бһҡбһҗбһҸбһҡбһјбһ”, бһҖбһ¶бһҡ Upload бһҡбһјбһ”бһ—бһ¶бһ– бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһҡбһјбһ”бһ—бһ¶бһ–бҹ”
* **`manager_dialogs.py`**: бһ•бҹ’бһ‘бһ¶бҹҶбһ„ Dialog бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҠбҹҒбһ”бҹүбһ¶бһҸбһәбһҳбҹүбһ„бҹӢ, бһҮбҹҶбһ“бһ¶бһү, бһҗбҹ’бһ“бһ¶бһҖбҹӢбһҡбҹҖбһ“, бһҳбһ»бһҒбһңбһ·бһҮбҹ’бһҮбһ¶ бһ“бһ·бһ„бһҖбһ¶бһӣбһңбһ·бһ—бһ¶бһӮбҹ”
* **`attendance_features.py`**: бһҖбһ¶бһҡбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһңбһҸбҹ’бһҸбһҳбһ¶бһ“ (бһҠбҹ„бһҷбһҠбҹғ бһ“бһ·бһ„бһҹбҹ’бһҖбҹҒбһ“бһҳбһ»бһҒ), бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚ бһ“бһ·бһ„бһҖбһ¶бһҡбһҳбһҫбһӣбһҖбһ¶бһӣбһңбһ·бһ—бһ¶бһӮбҹ”
* **`test.py`**: бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҖбһ¶бһҡбһңбһ¶бһҷбһўбһҖбҹ’бһҹбһҡбһҒбҹ’бһҳбҹӮбһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Tkinterбҹ”
* **`MyDatabase.txt`**: бһҹбҹ’бһӮбҹ’бһҡбһёбһ” SQL бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҸбһ¶бһҡбһ¶бһ„бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ бһ“бһ·бһ„бһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбҹҶбһҡбһјбҹ”

## бһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢ (Notes)

* бһҹбһјбһҳбһҖбҹӮбһ”бҹ’бһҡбҹӮбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢ MySQL бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ `db_utils.py` бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бҹ”
* бһҖбһ¶бһҡбһҹбҹ’бһҖбҹҒбһ“бһҳбһ»бһҒ (Face recognition) бһ‘бһ¶бһҳбһ‘бһ¶бһҡбһҡбһјбһ”бһҗбһҸбһҹбһ·бһҹбҹ’бһҹбһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбҹ’бһўбҹ”
* бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ—бһ¶бһҹбһ¶бһҒбҹ’бһҳбҹӮбһҡ бһҹбһјбһҳбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бһ–бһ»бһҳбҹ’бһ–бһўбһҖбҹ’бһҹбһҡбһҒбҹ’бһҳбҹӮбһҡ (Khmer fonts) бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

## бһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„ (Contact)

бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҶбһҺбһҪбһҡ бһ¬бһ”бһүбҹ’бһ бһ¶бһ“бһ¶бһ“бһ¶ бһҹбһјбһҳбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһўбҹ’бһ“бһҖбһҗбҹӮбһ‘бһ¶бҹҶбһӮбһҳбҹ’бһҡбҹ„бһ„ (Project Maintainer)бҹ”

бһҸбһҫбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҮбҹҶбһ“бһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ (Setup Guide) бһ”бһ“бҹ’бһҗбҹӮбһҳбһ‘бҹҖбһҸбһ‘бҹҒ?
